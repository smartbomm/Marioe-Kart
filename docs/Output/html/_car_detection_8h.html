◆ carDect1_execute()

uint8_t carDect1_execute ( )

Abfrage, ob an der CarDetectionUnit 1 seit der letzten Ausführung ein Fahrzeug erkannt wurde. Falls ja, wird dessen zugeordneter Regler gemäß obenstehender Tabelle berechnet.

Rückgabe
ID, bzw Reglerkanal, dem das erkannte Fahrzeug zugeordnet ist. Wurde kein gültiger Wert erkannt, ist die Rückgabe 99;
